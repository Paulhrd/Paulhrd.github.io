Creation of an optimized solution for the travelling salesman problem

×
The aim of the project was to create an optimized algorithm that searches for the shortest path through all the desired cities and back to the first city. First, we proved that the TSP (traveling salesman problem) is NP-complete, so we know that we can create a heuristic algorithm with polynomial complexity. We chose the TSP tabu search method to obtain an optimized path and then implemented it on jupyter notebook in python.
